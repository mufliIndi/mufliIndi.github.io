Pelajari Tempat Memulai!

HTML & CSS

Bahasa yang digunakan untuk membuat dan mendesain tampilan situs web Anda. HTML menambahkan struktur, dan CSS memberinya gaya.

jQuery

Library JavaScript yang cepat, kaya fitur, dan mudah digunakan yang menangani animasi dan permintaan Ajax.

Ruby

Bahasa tujuan umum yang dinamis yang sederhana dan produktif. Ini biasanya digunakan di seluruh dunia untuk membuat aplikasi web.

PHP

Bahasa skrip sumber terbuka yang dapat disematkan ke dalam HTML, dan cocok untuk pengembangan web.
